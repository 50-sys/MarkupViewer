AsciiDoc README File
--------------------

This is a modified & restricted version of the original AsciiDoc 8.6.9
distribution; and provides basic support for convertion asciidoc to html. +
For additional content you have to http://sourceforge.net/projects/asciidoc/[download]
and place files from archive to this folder.

Requirement
-----------
Empty file `MarkupViewer\asciidoc\__init__.py`.
